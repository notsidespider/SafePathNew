# SafePath v2.1

A disguised emergency alert mobile application built with React Native. SafePath allows users to silently send emergency alerts with their real-time GPS location to trusted contacts — all from an app that looks like an ordinary calculator.

## Features

- Disguised calculator interface for discreet access
- Emergency SOS alerts sent to pre-configured contacts
- Real-time GPS location sharing
- Encrypted contact storage
- Safe Haven map to locate nearby safe locations
- Personal journal for logging incidents

## Live Demo

> APK demo available on request. Repository includes full source code for local build.

## Getting Started

Make sure you have your React Native environment set up — see the [React Native Environment Setup guide](https://reactnative.dev/docs/set-up-your-environment).

```bash
# Install dependencies
npm install

# Run on Android
npm run android
```
